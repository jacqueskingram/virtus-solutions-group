# Virtus Solutions Group LLC Website

Static landing page for Virtus Solutions Group LLC.

## Files

- `index.html` - homepage content and structure
- `styles.css` - site styling and responsive layout
- `script.js` - mobile menu and footer year
- `assets/` - logo and favicon files

## GitHub Pages setup

1. Create a new GitHub repository.
2. Upload all files from this folder to the root of the repository.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save.
6. Add your custom domain `govirtus.net` in GitHub Pages settings.
7. Update DNS records at your domain provider based on GitHub's instructions.

## Notes

- The contact form uses `mailto:admin@govirtus.net`, which opens the visitor's email client.
- Once the LLC filing is fully approved, update the site as needed with finalized company details.
- Add phone number, address, capability statement link, EIN/UEI/CAGE/NAICS only after those are confirmed.
